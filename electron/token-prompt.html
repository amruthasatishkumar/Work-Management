🤖 AI Assistant Setup

The AI chat feature requires a GitHub Personal Access Token with model inference access (GitHub Models).

  1. Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Click Generate new token
  3. Set a name (e.g. "Work Management AI") and expiration
  4. No repository permission needed — just generate and copy the token
Please enter a token before continuing.
